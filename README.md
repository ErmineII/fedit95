# FEDIT95

Bitmap Font Editor

- Preview website is [here](https://nerry.jp/fedit95/src/)

## Requirement

- Electron

## Supported Format

- FONTX2 (binary) - import only
- FONTX2 (base64) - save / load
- (Original format) - coming soon?
